الرئيسية ٣ أنواع المستخدمين

أنواع المستخدمين

تقسيم المستخدمين يحدد صلاحيات الواجهة ومسارات الأمان: من يمكنه تغيير السعر أو حالة الطلب، ومن يمكنه فقط الشراء. هذا القسم أساس لسياسات تسجيل الدخول ولوحة التحكم.

زائر (Guest)

يمكنه تصفح المنتجات والبحث دون حساب. لا يُنشئ طلبًا مرتبطًا بهوية مستخدم إلا إذا سجّل أو أكمل بيانات اتصال وفق تدفقكم المختار لاحقًا.

مستخدم (User)

حساب مسجّل: تسجيل، دخول، تعديل بيانات، تغيير كلمة المرور، إضافة للسلة، إنشاء طلب، ومتابعة الطلبات السابقة.

مدير النظام (Admin)

صلاحيات كاملة على المنتجات، الطلبات، المستخدمين، المخزون، الخصومات، وعرض التقارير. لا يُفترض أن يظهر مسار الإدارة لغير المصرّح لهم.

يرتبط بـ

الخصائص الأساسية (٤)، لوحة التحكم (٧)، والتقارير (١٢).

معاينة — أعمدة الصلاحيات
زائر
  • تصفح المنتجات
  • البحث
  • لا لوحة تحكم
مستخدم
  • حساب + سلة + طلب
  • متابعة الطلبات
  • تقييم المنتجات
مدير
  • منتجات + طلبات
  • مستخدمون + مخزون
  • خصومات + تقارير
