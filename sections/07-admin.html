لوحة التحكم (Admin — متقدمة)

لوحة التحكم هي واجهة المدير لتشغيل المتجر يوميًا: إدخال منتجات، متابعة طلبات، ضبط مخزون، تفعيل خصومات، ومراقبة أرقام الأداء. النسخة «المتقدمة» في وثيقتك تشمل تقاريرًا ورسومًا تفصيلية أكثر من لوحة بسيطة.

لوحة المعلومات (Dashboard)

إحصائيات لحظية أو يومية: عدد الطلبات، إجمالي المبيعات، عدد المستخدمين. رسوم بيانية: مبيعات يومية/شهرية، أكثر المنتجات مبيعًا (تفصيل أعمق في القسم ١٢).

إدارة المنتجات

إضافة وتعديل وحذف، مع رفع صور وربط بالفئة والسعر والمخزون.

إدارة الطلبات

عرض التفاصيل وتغيير الحالة (Pending / Processing / Delivered / Cancelled) مع سجل زمني يُفضّل توفره للمساءلة.

إدارة المستخدمين

عرض الحسابات، تعليق أو إعادة تعيين كلمة مرور حسب السياسة، ومنع الاحتيال.

إدارة الخصومات والمخزون

ربط مباشر بالقسمين ٥ و ٦.

يرتبط بـ

التقارير (١٢)، الباقة Advanced (١٤).
